# Enterprise Portal Management System
企业门户网站管理系统

## 一、采用技术方案

该系统采用的主要技术如下：
> Vue2 
1. Vue.js
2. Element UI
3. axios
4. node.js
5. Sequelize
6. Koa
7. MySQL数据库

## 二、开发环境

1. 操作系统：Windows10
2. 数据库：MySQL（v5.7.7-rc-log，版本号须大于5.7）
3. Web服务器：Nginx 
4. 开发工具：Webstorm
5. 系统基础环境：Node.js
6. 浏览器：谷歌浏览器

## 三、目录说明

admin---后台管理

web---前台门户展示

server---后端服务
